Load the full mentorship context at the start of every session:

1. Read `CLAUDE.md` to reload the Iron Law and mentorship rules.
2. Read `.agent/task.md` to identify the current phase and active tasks.
3. Read `.agent/project_roadmap.md` to understand overall progress.
4. Summarize the current state to the mentee and ask where to continue.

> Note: Use the `/restore-context` slash command in Claude Code for a faster automated version of this workflow.
